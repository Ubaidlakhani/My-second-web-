What is html?

HTML

The hypertext markup language is the standard markup language for document design to be displayed in a web browser.

It defines the content & structure of web content. It is often associated by technologies. Such as Cascading style sheet & scripting languages. Such as java script.

It is developed by : WHATWG; World Wide Web Consortium

Initial released :1993; 31 years ago

Latest release : Living standard
